# WxLingua Documentation Assets

This directory contains images, diagrams, and other assets used in the documentation.

## Directory Structure

```
assets/
├── architecture/       # Architecture diagrams
├── screenshots/        # Application screenshots
├── diagrams/          # Database and flow diagrams
└── logos/             # Logo files
```

## Image Guidelines

- Use PNG for screenshots
- Use SVG for diagrams when possible
- Optimize images before committing
- Use descriptive filenames

## Adding Images

Reference images in markdown:

```markdown
![Alt text](assets/folder/image.png)
```

## Tools

Recommended tools for creating diagrams:

- [draw.io](https://draw.io) - Flowcharts and diagrams
- [Excalidraw](https://excalidraw.com) - Sketches
- [Mermaid](https://mermaid.js.org) - Text-based diagrams
